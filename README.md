# AI Biz (LocalAIModel)

Welcome to **AI Biz**, a next-generation Android application POC designed to give merchants complete, intelligent control over their business operations—all fully secured on-device.

## 🚀 Key Features and Benefits

### 🧠 On-Device AI (Gemma 3 1B)
AI Biz comes integrated with a powerful, fully local AI model to answer natural language questions about your business, eliminating the need to search through menus or reports.
- **Utmost Privacy:** Your financial requests never leave your phone. The model runs completely on-device.
- **Zero Latency:** Because everything happens locally, you receive immediate answers—even without an active internet connection.
- **Offline Reliability:** Have full view of your data and insights regardless of connectivity.

### 💰 Comprehensive Business Management
- **Transactions & Collections:** View precisely what you have collected today, yesterday, or this week.
- **Settlement Tracking:** Monitor settlements directly in the app.
- **Intelligent Navigation:** Ask the AI "Show my settlement status" and let the app securely fetch your data for you.
- **Dynamic Profile Analytics:** Detailed merchant profile management built purely on your locally processed patterns.

## 📱 Prerequisites
- Android Studio
- Android SDK
- Download the Gemma 3 1B on-device model file (`gemma3-1B-it-int4.task`) to load into the app on first launch.

## 🛠 Getting Started
1. Clone the repository: `git clone https://github.com/MechanicalMaster/LocalAIModel.git`
2. Open the project directory in Android Studio.
3. Sync the project with Gradle files.
4. Build and run the app on an emulator or physical device.
5. In the app, navigate to **Insights** and import your model file.
